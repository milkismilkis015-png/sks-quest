# SKS Quest — Геймификация для СКС Онлайн

## О проекте
SKS Quest — это модуль геймификации для клиентов сети ломбардов СКС. Проект превращает накопление бонусов в увлекательную игру с квестами, колесом фортуны и каталогом призов.

## Основные механики
- ✅ Ежедневный вход с серией (5 бонусов в день, +50 на 7-й день, +150 на 14-й)
- ✅ 3 ежедневных квеста (по 10-15 бонусов каждый)
- ✅ Колесо фортуны (1 раз в неделю, призы: 10-100 бонусов, скидки, бейджи)
- ✅ Каталог призов (8 призов разных категорий)
- ✅ Лидерборд (таблица лучших игроков)
- ✅ Уникальный помощник — воронёнок Золотце (подсказывает и мотивирует)

## Технологии
- HTML5, CSS3, JavaScript (Vanilla)
- LocalStorage для сохранения прогресса

## Как запустить
1. Скачайте или склонируйте репозиторий
2. Откройте файл `index.html` в любом современном браузере
3. Начинайте накапливать бонусы!
